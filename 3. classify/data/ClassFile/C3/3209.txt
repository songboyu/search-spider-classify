&nbsp;消化系统的变化 
　　当得知您已经怀孕之后，您就要面对十月怀胎的喜悦与烦恼了。女性在妊娠期身体的各个系统，包括呼吸系统、消化系统、泌尿系统等都会发生变化，以适应母体内的小生命。




　　消化系统
　　早孕反应和食欲改变
　　很多孕妇在孕6 - 1 0 周之间, 可有不同程度的恶心, 有时伴有呕吐，尤其于晨间起床时更为明显，这是 由于呕吐中枢对增多的雌激素暂时不能适应, 
或与精神因素有关。　&gt;&gt;&gt;&gt;怀孕时都有哪些征兆
　　早孕期间食欲和饮食习惯常有改变, 例如对 烟、酒、茶和油炸食物兴趣减退，而喜食咸、酸、冰食物和水果，甚至偏食，出现以上现象原因尚不明确。
　　口腔
　　牙龈充血, 水肿增生, 晨间刷牙时易牙龈出血。牙齿易松动, 过去认为这是缺钙所致, 现己知牙齿中钙质 是静止的, 不受血钙波动所影响。孕妇常唾液增多, 
唾液P H 值改变, 泛酸易造成孕期龋齿。 
　　专家谈：孕妇的口腔保健
　　胃肠
　　妊娠时随子宫增大, 胃被上举, 肠管被推向上方或腹腔两侧，盲肠和阑尾向外上方移位, 可达髂嵴水平, 一旦发生肠道和阑尾炎病变时腹痛不易与宫缩相鉴别, 
易延误诊断。
　　由于受大量激素影响, 胃肠道平滑肌张力减退, 蠕动减少, 减弱, 胃排空时间延长, 易感上腹部饱胀感, 故孕妇应防止饱餐。
　　孕妇肠蠕动减少, 
但小肠的吸收功能并无改变，对铁、钙由于体内的需要量增加，使吸收反见增多。粪便在大肠中积留时间长，水份逐渐被吸奴，使粪便干结而有便秘。便秘加上增大的子宫对直肠的压迫, 
以及性激素对血管壁平滑肌的扩张作用，常能引起痔疮或使原有痔疮加重。&gt;&gt;&gt;&gt;准妈妈孕期当心痔疮
　　肝和胆囊
　　妊娠期肝脏改变不多，未见增大，血流量未见增多，肝细胞形态大小虽略有改变，但并无异性。肝功能试 验略有改变; 
胆固醇、碱性磷酸酶、总胆质增加。血浆白、白蛋白/ 球蛋白比值减少。

皮肤的改变
　　色素沉着
　　妊娠时皮肤色素沉着以面颊、乳头、乳晕、腹白线、外阴等处较为明显。面颊部可见 呈蝶形分布褐色斑，称妊娠斑, 于产后逐渐消退。 
　　妊娠纹

　　孕妇腹壁、大腿等处皮肤可见长5--6cm 、宽约0.5cm的不规则平行裂纹, 这些纹呈紫红或 淡红, 产后逐渐变为银白色, 持久不退。一般认为: 
是增多的肾上腺皮质激素参与而发生。 &lt;如何避免讨厌的“妊娠纹”&gt; 
　　毛发
　　妊娠期睾酮和肾上腺皮质激素产生增多, 可使毛发, 主要是阴毛、腋毛增多、增粗, 头发卷曲度减少, 偶 有前顶头发减少者。
　　腺体
　　妊娠时汗腺、皮脂腺功能亢进, 腋下和会阴部皮肤皱摺处可有炎症而引起糜烂, 常不易治愈, 需在产后才消退。
　　搔痒
　　搔痒是较常见症状，可局限于腹部或外阴，有时全身都有搔痒，其原因不明。

心血管
　　心脏工作量增加
　　由于体内血容量增加, 心脏工作量也加重, 尤其在妊娠末期。由于隔肌上升心脏向前移位, 心率每分钟增加10--15次, 妊娠13--23周最为显著, 
每次心脏收缩所排出的血量称心搏量可比非孕期增加30%,平均值约80ml/次。
　　血压
　　妊娠对动脉压影响较少, 收缩压几乎不受影响；舒张压于孕中期约下降10mmHg(1.3 KPa)；孕晚期恢复原有水平。而体位改变可影响血压测定值, 
坐位血压较高,仰卧时较低,左侧卧位时更低。
　　由于孕妇外周血管扩张, 故孕妇虽心排出量增加而血压无变化, 而能使四肢血流量增多, 手的血流量是非 孕的7 倍, 足的血流量是非孕2 - 3 
倍,使孕妇手(足)掌呈充血状, 有温热感。

内分泌系统
　　垂体前叶增大 
　　垂体前叶增大1 - 2 倍，嗜酸细胞增多肥大成为所谓的"妊娠细胞" ，有时增大的垂体可压迫视神经交叉而致两颞侧偏盲。产后随垂体缩小而于产后1 0 
天左右恢复。如产时或产后出血性休克，可使增生 肥大的垂体缺血坏死，导致席汉氏综合征。 
　　妊娠期垂体前叶分泌的泌乳素( PRL ) 、促甲状腺素( TSH ) 、促肾上腺皮质激素( ACTH ) 和 黑色细胞刺激素( MSH ) 
增多。PRL 于孕7 周起开始增多，血清中PRL 值可由非孕时的10mg/ml上升为足月妊娠时的200mg/ml, PRL 
有促进乳房发育的作用，为产后哺乳作准备。
　　哺乳者在产后8 0 - 1 0 0 天或更长时内降到非孕期时水平。未哺乳者在产后3 周内降到非孕期水平。TSH 和ACTH 的分泌 
增加但无症状改变。MSH 的分泌增多, 使面颊、乳头、腹白线、外阴等处皮肤有色素沉着。 生长激素( G h ) 的分泌无改变，促性腺激素( FSH 、LH ) 
的分泌减少, 故妊娠期卵巢中的卵泡不 再发育成熟, 也无排卵。而垂体后叶所产生的催产素和加压素无改变。 
　　甲状腺甲状旁腺
　　妊娠后由于受多量雌激素影响, 血浆中结合型的T3 、T4 增多，而游离型的T3 、T4 减少, 通过负反 馈作用, 使垂体产生甲状腺素( TSH ) 
增多, 刺激甲状腺使之均匀性增大, 比非孕时增大6 5 % 。 甲状旁腺在妊娠时呈增生状态，孕2 4 周起甲状旁腺素的浓度逐渐升高。机理尚不清楚。
　　胰腺
　　妊娠期胎盘滋养层细胞所产生的胎盘生乳素、雌激素、孕激素等，在外周组织中, 都有拮抗胰岛素的功能, 能破坏胰岛素的酶, 
故孕妇是以增加胰岛素的分泌来维持其体内的糖代谢, 表现为胰岛的β细胞功能亢进, 在 孕中期即开始, 接近预产期时达高峰。如胰岛的代偿功能不足, 
不适应这一改变，将于妊娠首次出现糖尿病, 称妊娠糖尿病。
　　肾上腺
　　由于妊娠受的大量雌激素的影响, 肾上腺皮质所分泌的激素增加, 每日分泌量由非孕期的139 nmol ～ 695 nmol(50 ～ 250 ug) , 
上升为足月时2780 nmol (1mg) 左右, 增为原来的3 倍。 
肾上腺髓质所产生的肾上腺素和去甲肾上腺素都无改变，仅于临产后由于对子宫收缩的应激反应而增多。

血液的变化
　　孕妇的血液比非孕期妇女增加约两公升(2000ml), 这些血液供应胎儿所需的氧气和养份。自孕 6 周起, 母体血容量开始增多, 
孕32--34周时达高峰。血容量增加包括血浆与红细胞的增加, 但骨髓则不断产生红细胞, 红细胞主要成份血红素含有铁质, 为了制造妊娠期所需的额外血红素, 
孕妇运用自身储备的铁质, 如储备铁质不够, 即发生贫血。因此妊娠期贫血多为缺乏制造血红素的铁质, 血红素不够也就不能制造额外的红细胞。
　　复合维生素B中含有叶酸,叶酸也是制造血红素的成份,如果缺乏也会引起贫血,为了避免贫血,孕妇的食物必须含有适量铁质和叶酸。而叶酸的补充应从孕前就开始。

　　呼吸系统
　　由于子宫逐渐增大,隔肌上升,上升幅度约4 c m。肺披挤压上去, 所以孕妇容易感到喘不过气,也会感到 晕眩。这是妊娠期常见现象, 也是正常的。 

　　肾脏和泌尿系统
　　孕妇因为妊娠生理变化, 肾血流量增加。早期由于子宫扩大挤压膀胱,尿频显著。孕后期, 随胎头下降入 盆也会挤压膀胱, 
造成同样结果。由于肾血流量及肾滤过率增多,葡萄糖滤过增多,当肾小管对葡萄糖的再吸收 不能相应增加时,即可出现糖尿,约有1 5 % 
孕妇有糖尿、氨基酸滤过增加,但无蛋白尿,由于以上排出物增加,为 细菌生长提供了物质条件, 故孕妇易发生尿路感染 
　　体重增加
　　妊娠期母体体重增加可达 12 Kg左右，一般在孕13周前体重加不明显, 
孕13周后平均每周增加350g,直至足月，但每周体重增加在500g以内，如超重要注意有否隐性水肿。
