　　“工作的幸福感是一样的，而工作中的不幸却各有千秋。”人人都在追求幸福生活，殊不知，工作本身是否让你感觉幸福，占据了你对人生幸福感的一半。　　在51JOB的职场沙龙上，向阳生涯职业规划专家有幸以“职业规划师”的身份作为嘉宾共同参与了关于“工作幸福”的主题沙龙讨论。而对于职业幸福感，向阳生涯职业规划专家团队长期以来也一直在关注并研究着。沙龙之余，我们想通过本文能够对更多人的提供相关的帮助。　　《职业规划岗位资格证书》专家组秘书长、向阳生涯职业咨询机构首席职业规划师窦吉先生指出，在2005年第一季度对近千例职业人士的 “工作幸福感”调查中发现，当前职场人士的工作幸福感普遍偏低，整体工作幸福指数仅为10（最高20分，最低0分）。只有10％的被调查者幸福地工作着。处于中等偏下状态。分析同时显示，上海、北京等经济发达城市职业人的幸福感要比杭州、成都、西安等城市的职业人工作幸福感差。　　幸福的感觉直接影响着我们的工作效率、影响着我们的个人发展。下面请让我们来跟着一组向阳生涯职业咨询机构咨询中遇到的情况来一起分析如何让你的工作幸福感节节提升。　　没有激情的机器人　　 每天早晨，Ivy睁开惺忪的双眼，一想起即将面对的工作，痛苦和无奈就涌上心头。又要面对重复单调的秘书工作，让她对每天的工作不再充满激情和想象。作为一家贸易公司的经理秘书，Ivy已经作了将近4年了，每天她都在重复而机械地整理资料、联系客户、制定日程…… 已经没有新鲜感和挑战性，成就感就更加无从谈起。在谈及“你在工作中是否感受到幸福呢？”时，Ivy说，“如果让你像机器人一样程式化地做某些事情，你会幸福吗？你会感觉自己的价值吗？”　　 向阳生涯职业咨询机构职业规划师洪向阳分析：工作中的被认同和小有成就让职场人士轻舞飞扬般地美美陶醉一把，是提升工作幸福指数的良药，能够化解工作中琐碎的烦心事和倦怠感。职场如战场，不进则退。保持对工作成绩的追求，可以使你的职场幸福感和职场竞争力节节攀升。　　像Ivy这样的职业厌倦者，不妨及时与上司进行沟通，让上司了解你渴望挑战希望提升能力的想法，同时在适当时机提出自己的工作要求，希望能得到上司的认可，并委以重任。同时应该注意，新的工作机会应该与以前的工作经验有共通和关联性，这样才能成就更长久稳定的职业生涯。　　看见电脑就“晕机” 　　工作刚过一年的陈新这样形容自己的工作，“我的工作可以分解为上班----刷卡----开电脑----上网、聊天-----吃中午饭-----开电脑----上网、聊天-----下班。上班后打开所有能用的聊天工具，在各个论坛上潜水灌水就是我的工作。”在一家科研所工作的陈新一脸无奈地说。天天的生活一成不变，现在一看到电脑他就有晕机的感觉。“‘工作’对我来说已经是很陌生的词语了， 基本上无事可做，没有压力，没有挑战性，只是把自己的意志和朝气消磨殆尽了。”没有工作，何来所谓的工作幸福感呢？
　　《职业规划岗位资格证书》专家组秘书长、向阳生涯职业咨询机构首席职业规划师窦吉先生分析：充实是工作幸福的保障，无所事事的工作幸福感是无本之木、无源之水。不能找到自己恰当的工作位置，在新的工作环境中无所适从，进而无所事事是许多出入职场的新人经常碰到的问题。工作中最大的悲哀莫过于无所事事，虚耗时光。相反，忙碌而充实的工作职场人士充满成就感和自身能力的肯定。　　　　如果你现在工作很闲适，规划师建议你开始考虑“工作任务不找上门，那就自己主动去找工作任务”，尽量多、快、好地熟悉公司业务，找到工作上的空缺机会和任务。让自己在忙碌中充实、充实中提升不失为经营工作幸福的捷径。　　不作上司们斗争的炮灰 　　“职场如江湖，人在江湖身不由己”。数天前，二把手在没有跟一把手商量，也没有明确告知廖建这个部门主管的情况下擅自安插了一个司机编制进了廖建的部门。廖建主动跟二把手沟通，表明了自己的观点：随意安插进人不利于公司制度的建立和管理。二把手直拍胸脯向廖建承诺不会有问题的。没想到，新来的司机去一把手那里签文件进公司之际，却受到了一把手的“刁难”，并狠狠批评了廖建一番，言语之间无外乎指桑骂槐、杀鸡儆猴之意。但是，无意之间，成了上司发泄彼此不满的炮灰让廖建对“职场游戏”心有余悸。　　向阳生涯职业咨询机构职业规划师洪向阳指出：工作幸福感指数调查分析表明，影响工作幸福感的一大杀手是所在单位的压抑的工作环境和错综复杂的人际关系。融洽的人事关系，营造的是一种细水长流的幸福氛围。就职场人士来说，对这种环境的依赖和感受，是积久日深的。 　　前段时间，网上流行说每个办公室都在上演不同版本的《金枝欲孽》：扮天真、扮无知、扮无辜、搏取同情如玉莹者，扮与世无争，冷静处事如尔淳者，安分守己如安茜者，圆滑处世如孙柏杨者，霸气十足、处世不惊如如妃者，比比皆是。 如何让自己在纷繁复杂的人际网中游刃有余，营造融洽的人际关系是许多职场人士的困惑所在。　　向阳生涯职业规划专家建议大家不妨奉行“古钱币”的处世之道，即外圆中方――外表圆滑世故、内在充满正气和同情心。这样在公司管理制度的不健全的情况也能在保护自己的基础上坚持原则。　　幸福需要自己把握 　　观点：职业规划专家金培华―《职业规划岗位资格证书》、向阳生涯职业咨询机构职业规划师指出：成就感、挑战性、对工作的热爱与否、人际关系状况、工作环境、管理制度和文化氛围都是影响工作幸福感的重要因素。因此，作为职员，我们应该在力所能及的范围内努力提升自身对工作的幸福感，比如抓住机会提升成就感、塑造合作融洽的人际关系、勇于面对挑战等等。一个让你深感幸福的工作必定带给你一个幸福的生活状态，那么为什么不让你的工作充满乐趣呢！　　职场幸福指数测试 ――本测试由上海市《职业规划岗位资格证书》培训认证职业规划师专家组开发　　说明：本测试目的旨在帮助大家分析自己当前的职业满意状态，并帮助大家找出原因，以期及时调整。作题时需要大家抱着对自己负责的态度，诚实作答即可。　　题目 　　选项 1、我很热爱现在的工作。 是 ？ 否 　　　　　2、在工作中我很有成就感。 是 ？ 否 　　　　　3、上司能及时了解我的需求 是 ？ 否 　　　　　4、我与同事间的关系很融洽。 是 ？ 否 　　　　　5、公司在工作时间安排和任务分配方式上很人性化。 是 ？ 否 　　　　　6、我每天过得都很充实。 是 ？ 否 　　　　　7、公司有良好的激励机制。 是 ？ 否 　　　　　8、我所从事的工作与自己的期望值和能力相匹配。 是 ？ 否 　　　　　9、公司有很灿烂的远景，让我觉得工作有目标。 是 ？ 否 　　　　　10、我对公司有一种家的感觉。 是 ？ 否　　得分方法 ：答“是”得2分；“？”得1分；“否”得0分。将各题得分累加，算出自己的总得分。 　　得分为0～3者：你的工作满意度极差，在工作中你无法获得幸福感，你很有必要找个职业规划专家或比较有资质的职场人士为自己把把脉，重新勾画和设计一下自己的职业规划，调整一下自己的生活方式，以期让日子过得感觉好起来。　　 得分为4～7者：你的工作幸福感较差，工作状态不怎么样，这让你容易沮丧，情绪低落。你不妨检讨一下自己的状态，看看是不是目标太高，过分追求完美，或是自卑感什么的让你工作难以开展。变变想法，也许感觉会有改变。　　得分为8～17者：你的生活状态一般，有喜有忧的日子使得你和多数人一样。
　　得分为18～20者：你有相当高的职业满意度。你不一定就是富人或有地位的人，但你的心态很好，一个人能感到幸福是件不容易的事，在这里我们只能向你表示祝贺，并希望你永远幸福。　　在此，向阳生涯职业规划专家团队也希望所有的职场人都能像这类人一样能够幸福的工作着，如果你得分在7分以下，建议你尽快寻求问题解决的方法，向阳生涯职业规划专家将全力帮助你排除障碍，支持你迈上职业发展的幸福快车道。（完　　上海向阳生涯管理咨询有限公司汇集了国内顶尖的职业生涯管理专家与国家职业心理咨询师专家团队，专注于个人与团队“职业生涯规划”的理论研究和实践指导；是上海紧缺人才办公室唯一授权进行《职业规划岗位资格证书》认证培训的单位，中国首批职业规划师的培训机构。　　向阳生涯职业规划专家以帮您进行专业的职业定位、转型、职业规划发展等为已任，以助您成功求职、获取高薪、高职及职业生活的平衡发展为宗旨。职业成功，规划先行；专业咨询，政府公认，是您职业成功的有力保障。
