　　【eNet硅谷动力消息】美国东部时间4月11日（北京时间4月12日）消息:苹果公司在星期二推出了其最新版本的远程遥控管理软件Apple Remote Desktop 3。苹果公司称，这个新版本管理软件除了与Mac OS X操作系统更紧密结合在一起之外，还是一个通用于PowerPC苹果电脑和基于英特尔处理器苹果电脑的应用软件。
　　新版本软件中包括许多功能，用户可以在多个Tiger系统中进行Spotlight搜索。Spotlight是苹果公司开发的一款内建式桌面搜索引擎，用户利用它可以在文件中搜索文件名和文字信息。
　　苹果公司还在新版本软件中加入了30个自动控制操作来自动重复执行系统管理任务，同时新增了一个Dashboard控件，用户利用这个控件可以对远程系统进行监视；软件中还增加了一个自动安装功能，当用户移动系统重新联机的时候就会自动安装软件。
　　之前版本的Apple Remote Desktop软件的用户将会欣喜地发现新版软件中在本地与远程计算机进行文件转移操作时增加了一个远程拖放功能，大大方便了用户的操作。而且在本地与远程计算机之间还增加了远程复制与粘贴功能。
　　10系统版的Apple Remote Desktop 3软件的售价为299美元，无限系统版Apple Remote Desktop 3软件的售价为499美元。
