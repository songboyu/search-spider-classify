　　CNET科技资讯网4月12日北京报道（文/王丹） 4月7日，微软面向中国市场发布了Windows Live Messenger测试版，同时宣称“MSN Messenger在中国本土的注册用户已超过1050万；Windows Live Messenger的推出也标志着整个Windows Live产品线的开启，它将给用户带来全新的Internet 体验，也是互联网革命性的一步。”
　　根据市场调查公司艾瑞提供的数据，中国25到30岁的网民，MSN用户占到41％，相比腾讯QQ，MSN用户成熟度更高。而从用户受教育程度来看， MSN用户58％拥有大学本科以上学历； MSN用户月均网上消费是127元。
　　作为微软成功的互联网品牌，MSN过去几年来飞速发展，成为很多商务人士不可或缺的工作、生活工具。去年10月微软推出了全新互联网平台战略，Windows Live、Office Live 和Xbox Live将构成微软未来的网络服务平台，这引起很多人关于MSN将退出的猜测。
　　4月11日下午5点，在北京MSN装修一新的办公室，CNET科技资讯网记者访问了微软（中国）公司副总经理兼MSN事业部大中华区总监罗川，他谈到了Windows Live服务的理念、对MSN的深厚感情、是否会给“Live”取中文名字，以及如何携同合作伙伴抓住互联网广告带来的丰厚利润……
　　Q：Windows Live到底是什么?
　　Windows Live有一个设计理念：让所有服务都以我们每一个个人为中心的，它首先是一个“关系”的服务，比如朋友和家人。 Windows Live的目的是加深你和你所关心的人之间的关系，帮助用户快速准确地沟通信息。这些信息放在不同的地方，可以用桌面搜索找到，也可以用搜索引擎找到，或者用RSS不停地推给你。
　　另外，我们也希望indows Live能够成为一个让合作伙伴能够在上面开发应用的一种平台，就像今天的Windows 操作系统，在他上面难免，构建了一个健康、庞大的独立开发商体系，比如金山、用友、金蝶这样的公司也是跟着Windows的成长一起发展的。
　　Q. Windows Live如何把Windows 的成功经验移植到网上？或者说Windows Live的商业模式是什么？
　　和以往Windows、Office的软件商业模式不同，Windows、Office通过许可来获取效益，缔造了一个软件帝国，而未来，Windows live则将主要依靠广告，向最相关的人提供服务来推动广告，由此获得商业价值。
　　今天互联网受众群所需要的既有信息，也有社会关系，还有追求爱好和兴趣。互联网几乎比任何其他媒体更能够把用户有效粘连到一个信息网络中来。把广告商的信息最恰当地提供给用户，这就完成了一个广告配送和发布的最有效过程。事实上Google在这方面已经做得非常先进，我们希望Windows Live能够赶上它的步伐。
　　Q：Windows Live推出以后，MSN是不是就退出了？
　　人们希望获得两种信息，一种是编排过的，另外一种是用户选择。MSN.com是广泛的、经过编排的内容的体现；而Live.com是用来进行选择的一个网站。这是两个不同的入口。
　　另外，MSN是做一个服务，而Windows live是做一个平台，将来在它上面可以做更多的增值开发，比如可能开发出很多应用接口，用户可以在上面开发机器人服务。这样的生态环境使得整个互联网的用户、合作伙伴、广告商、开发商都能够受益。
　　Q：这听起来很有意思。中文版的Windows Live是今年5月份发布？
　　它的发布以后不会有一个特定的日子，因为软件服务跟软件不一样。某个软件如果说12月发布，那么1月份时可能到处都能够买到了。但是互联网的推广更多是靠口口相传的，一个用户在试用，很快就会把这个消息传递到很多用户中去。
　　Q：您前面提到中国MSN没有大量投入就取得显著增长，而Windows Live下半年将有一个很大的宣传，具体推广计划是什么？
　　现在推广计划正在酝酿当中，会有三个方面的重点，一个是Windows Live搜索，另外一个是社会网络系统，包括即时通讯、共享空间、电子邮件；第三我们还希望在广告平台上花很大力气，但广告平台有可能不能很快推荐出来，更多的是在研发和了解用户需求上。外国很多广告平台进入中国都遇到很多的挑战，比如雅虎、Google。
　　所以如何把广告系统引进中国，首先面临的是广告商还不太成熟；其次，网络营销基于一个很好的信用系统，而目前连个人信用卡使用比例都很少更不用说公司。广告系统像一个电子商务，如何实现自动化的广告发布还需要相当长的时间。
　　Q：微软一年有450亿美元收入，但是起码目前网络广告的市场还比较小，等它成长到能够承载450亿美元收入的规模，是不是为时尚早？如果把未来都压在这上面，是不是有一点点危险？
　　罗川：广告市场其实不小，以全球来看，搜索加上横幅式广告，到2008年将会是200亿美元的市场。广告产业本身比软件产业大很多。现在互联网广告只占到整个广告市场的2.5％左右，所以你把200以乘以40，整个全球广告业的市场是很大的。
　　Q：微软经常说微软赚1块钱，合作伙伴赚8块钱，你把自己的业务挪到广告平台上来，合作伙伴从哪里赚钱？
　　Windows Live.com，会提供很多增值服务，比如天气预报，很多人看天气预报的时候就有很多广告机会，合作伙伴可以选择微软，也可以选择Google，他只要有这样的广告平台接触的时候，可以把这些容量变成他的商业，变成他的现金。很多人都使用Live.com，如果有一百万人每天都来看这样的一个服务，一百万人就算每天看4页，就有4百万的配给率，流量可以变为收入。现在互联网最缺乏的就是如何挣钱，如果有这样的推动手段，有了好的广告平台，就可以使第三方的合作伙伴在上面好的收益模式。
　　Q：那他们在Google的平台上开发如何？
　　任何软件公司都可以开发一个跟Windows Live一样的平台，这是技术的竞争。Google也可以开发一个这样的Live，百度也可以，但开发Live.com这样的软件，投资上确实很大，比如可能需要上千名工程师做一两年的时间。
　　Q：在Windows Live的推广过程中有哪些阻力？
　　很多阻力。很多人都已经把MSN作为一个动词了，现在MSN 突然变成了Windows Live的内容服务，不要说用户，我们的员工也有很多感情上的难以割舍。这种感情上的阻力，不仅在中国，在其他的国家也有，比如像西班牙、荷兰，很多孩子是没有电子邮件的，一个孩子找到另外一个孩子都是通过即时通讯、Spaces。
　　但是这个品牌的变化事实上并不会带来用户体验的下降，Windows品牌比MSN品牌要强势很多，而“Live”代表活力，如果Windows能够充满活力，用户也许会产生不同的体验。
　　Q：会不会给它取一个中文名字，“Live”这个单词可能有些人不认识。
　　这是一个很好的建议，我们一直在考虑要不要取一个中文名字，很多企业都用中文名字，比如可口可乐，但互联网企业，比较少用中文名字。这个问题还没有结论。
　　另外互联网人口庞大，相应来讲都有一定英文知识。此外，我个人觉得我们应该更国际化，企业也需要国际化，文化也需要国际化,我们在文化心态上是不是可以更开放一点容纳更多的东西。
　　Q：Windows Live这两个词比较复杂一点，不象“IBM”那么简单。也许你可以跟国家语委商量一下，造一个外来词；语言也是在不断地发展中。
　　当初我们觉得MSN这个词很难念，很拗口，但后来也慢慢习惯了。所以我们还没有一个最后的结论。这个问题也存在于韩国市场。不过不是Windows的品牌，主要是Live这四个字母的品牌，如何把它推出去，这是一个有挑战性的事情。
　　Q：既然Windows Live成立一个平台，并且又有了Office Live、Xbox Live，是不是以后我们的PC就可以不要Windows操作系统了？
　　这是一个非常有趣的话题，有两个东西少不了：第一要调用内存，第二要管理硬盘，这离不开操作系统。
　　Q：这些Linux也可以做到。
　　Windows Live提供的是一个互联网用户体验的环境，有很多标准的应用，这些应用要使得你下载一个歌曲的时候能够听到，下载一个视频能够看到，下载一个文件能够读到，这些东西都完全依赖互联网为你提供信息源，但在本地需要有应用程序。未来不是消亡掉其中的一个，可能你会选择其中的一个。
