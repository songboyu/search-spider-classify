　　摘要：据外电报道，软件巨头微软公司发布了 SQL Server 2005的升级版本，在它的数据库软件中增加了一些新的功能。
　　天极网4月20日消息(他山石 编译)据外电报道，软件巨头微软公司发布了 SQL Server 2005的升级版本，在它的数据库软件中增加了一些新的功能。
　　本周三微软公司表示，新发布的 SQL Server 2005 Service Pack 1 将包括新的镜像工具，并密切了与SAP 公司NetWeaver软件的联系。微软公司还同时升级了它的SQL Server 2005的免费版本。
　　微软公司期待已久的SQL Server 2005在去年十一月份投放市场，它将和甲骨文公司以及其他对手的数据库产品展开激烈的竞争。
　　在Service Pack 1中，将包括对SAP公司 NetWeaver商务智能软件的链接，客户可以 输入SAP公司商务信息库中的数据，能够更容易的对报告进行创作和管理。新的升级还包括数据镜像，它的目标是在帮助灾难恢复时向用户提供数据快照。
　　在微软公司数据库产品免费版 SQL Server 2005 Express Edition 中同样增加了新的功能，其中包括全部短信搜索和一款简单的绘图管理工具。(完)
