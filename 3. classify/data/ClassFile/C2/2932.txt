　　――Windows Mobile应用开发大赛举办游戏开放日
　　4月22日，在全国有着广泛影响的2006 Windows Mobile应用开发大赛暨Windows Mobile游戏开放日再次走入国内著名的游戏开发教育机构――游戏学院CBD校区，与游戏学院为大赛专门成立的Windows Mobile开发小组直接交流，提供有力的技术支持。
　　4月7日，Windows Mobile游戏开放日第一次走入游戏学院，便与游戏学院的学员进行了面对面的交流与互动，使学员们更直观地认识了Windows Mobile智能手机和Windows Mobile游戏开发的全过程。游戏学院专门成立Windows Mobile游戏开发小组，在培养学员动手操作能力的同时，开发Windows Mobile操作系统下的精品手机游戏，使学员紧跟技术前沿和市场发展，更加符合游戏企业对手机游戏开发人才的需求，为中国原创应用和应用开发群体的壮大增添动力。
　　据了解，本次2006 Windows Mobile应用开发大赛是由信息产业部软件与集成电路促进中心（CSIP）携手微软（中国）公司共同推出的。继微软2004和2005两年成功举办 Windows Mobile应用开发大赛后，经过一年的酝酿， 2006 Windows Mobile应用开发大赛于3月1日正式拉开序幕。据参与本次大赛组织的MII-Microsoft嵌入式联合实验室负责人郑瑜介绍说，为了加强对中国本地开发者的技术支持，促使Windows Mobile技术的本土化普及，本次大赛期间将举办一系列技术培训活动，为此，大赛主办方进行了大量的准备工作，在比赛期间推出了包括“Windows Mobile开发黄金周”在内的一系列培训活动，并通过大赛网站定期推出Windows Mobile技术开发资料，让参赛者更方便的学习Windows Mobile应用开发技术。
　　此次在游戏学院举办的Windows Mobile游戏开放日就是这一系列活动中的重要环节。活动期间，来自微软公司、CSIP、以及微软合作伙伴的技术专家为学员们全面讲解和展示了Windows Mobile游戏开发技巧，并现场演示了最新的Windows Mobile移动终端，帮助学员们掌握开发技巧，为参加本次大赛奠定了良好的技术基础。
　　据微软相关部门介绍，微软一向重视移动游戏领域。早在2004年由微软举办的第一届Windows Mobile应用开发挑战赛上，就由于参赛游戏作品表现突出，特意将游戏应用单独作为大赛的评奖类别。而本届2006年度Windows Mobile应用开发大赛更是将游戏应用列为正式竞赛类别。此次大赛主办方与游戏学院联合举办的Windows Mobile游戏开放日活动更是直接促进了移动游戏应用的培训与开发。
　　Windows Mobile在这几年中经历了迅猛的发展。目前在全球已经有47家手机制造商和55个国家的102家移动运营商推出了基于Windows Mobile的手机；可供销费者选择的Windows Mobile手机超过了100多款。最新的Windows Mobile 5.0更是为娱乐和商务应用提供了更加强大的支持。在中国也有越来越多的手机厂商和应用开发商与微软合作推出Windows Mobile的手机和应用。目前在中国已有800多家独立软件开发商在从事基于Windows Mobile的应用开发。微软一直以来都在利用公司强大的技术力量为国内的应用开发者提供广泛的支持和培训。此次大赛以及其中的Windows Mobile游戏开放日等一系列活动为国内广大致力于Windows Mobile游戏应用开发的人士提供了一个充分学习与交流的机会。
　　据信息产业部最新公布的数据，2006年我国将新增手机用户4800万户，手机用户总量有望达4.4亿户。而其中手机中的游戏功能也越来越赢得更多的用户而呈现出巨大的市场商机。摆脱了线缆的束缚，具有随时、随地、随身的特点，更适合人们在移动中休闲和娱乐，今后手机游戏市场将远远大于网络游戏市场。
