1. 安装 python-2.7.9.msi（自带pip）
   其他版本可运行 "python get-pip.py" 安装pip（建议2.7版本，其他版本可能造成不兼容）

2. 将 安装目录、安装目录\Scripts 加入系统环境变量
   如 C:\Python27;C:\Python27\Scripts

3. 每个项目下的build-env-*.bat，即为该项目的环境搭建脚本。

注：
1. 建议使用Sublime Text 2作为编辑器，（Sublime Text 2.0.2 Setup.exe）
   可将其加入右键快捷菜单，（安装时勾选Add to explorer context menu）

2. 运行build-env-*.bat时，需要联网下载一些数据包
